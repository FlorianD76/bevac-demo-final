# BEVAC Engineering — Site démo

Maquette de site web pour **BEVAC Engineering SPRL**, bureau d'études indépendant en sécurité et signalisation des transports (Braine-l'Alleud, depuis 1964).

🔗 **Démo en ligne** : https://floriand76.github.io/bevac-demo-final/

## Contenu

- Page unique présentant les services, domaines d'application, approche et références de BEVAC
- Logo officiel intégré en SVG vectoriel (header + footer)
- Galerie photo issue de réalisations terrain
- Site responsive (desktop / mobile)

## Stack

HTML / CSS / JavaScript statique, hébergé sur GitHub Pages.
